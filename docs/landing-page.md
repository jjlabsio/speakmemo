## 2. 섹션별 콘텐츠

### 2-1. 네비게이션

- 좌측: semo 로고 (텍스트 로고)
- 우측: **[얼리액세스 신청하기]** 버튼

### 2-2. 히어로 섹션

**메인 카피:**

> 그냥 말하세요.
> semo가 알아서 정리해 드릴게요.

**서브 카피:**

> 녹음 버튼 하나로 시작하세요.
> AI가 요약, 핵심 포인트, 할 일 목록까지 자동으로 만들어 드립니다.

**CTA 버튼:** 지금 말해보세요

**비주얼:** 앱 스크린 목업 또는 녹음 → 정리 결과 변환 애니메이션
(Phase 0에서는 정적 일러스트/목업, 앱 완성 후 실제 스크린샷으로 교체)

### 2-3. 핵심 기능 소개

3가지 핵심 기능을 카드 형태로 나란히 배치:

| 기능          | 제목                         | 설명                                                      | 아이콘 방향       |
| ------------- | ---------------------------- | --------------------------------------------------------- | ----------------- |
| 음성 녹음     | 말하기만 하면 끝             | 타이핑 없이 녹음 버튼 하나로 메모를 시작하세요.           | 마이크            |
| AI 자동 정리  | AI가 깔끔하게 정리           | 요약, 핵심 포인트, 할 일 목록까지 자동으로 만들어 드려요. | 문서 + 체크리스트 |
| 구조화된 노트 | 다시 듣지 않아도 한눈에 파악 | 구조화된 노트로 언제든 빠르게 확인할 수 있어요.           | 노트 / 목록       |

### 2-4. 사용 흐름 (How it works)

3단계로 표현:

| 단계 | 제목          | 설명                                             | 비주얼 방향         |
| ---- | ------------- | ------------------------------------------------ | ------------------- |
| 1    | 녹음하기      | 녹음 버튼을 누르고 말하세요. 3분이면 충분해요.   | 녹음 버튼 + 파형 UI |
| 2    | 잠깐 기다리기 | AI가 내용을 듣고 정리해요. 몇 초면 완료됩니다.   | 로딩/처리 중 UI     |
| 3    | 확인하기      | 요약, 핵심 포인트, 할 일 목록을 바로 확인하세요. | 완성된 노트 카드 UI |

### 2-5. 사용 시나리오

타겟 사용자의 공감을 끌어내는 시나리오 카드:

| 시나리오          | 문구                                                                |
| ----------------- | ------------------------------------------------------------------- |
| 출퇴근길 아이디어 | 걸으면서 떠오른 아이디어, 놓치지 마세요. 말하면 semo가 정리해 줘요. |
| 회의 끝난 직후    | 회의 끝나고 정리할 시간 없죠? semo가 핵심만 뽑아 드려요.            |
| 자기 전 메모      | 잠들기 전 떠오른 생각, 타이핑 대신 말하세요.                        |

### 2-6. 얼리액세스 CTA

**제목:** 가장 먼저 사용해 보세요

**설명:** semo는 곧 출시됩니다. 이메일을 남겨주시면 가장 먼저 초대해 드릴게요.

**폼:** 이메일 입력 + [무료로 시작하기] 버튼
(Tally 또는 Google Form 임베드)

### 2-7. FAQ

| 질문                             | 답변                                                                                          |
| -------------------------------- | --------------------------------------------------------------------------------------------- |
| semo는 무료인가요?               | 네, 무료로 사용할 수 있어요. 추후 더 많은 기능이 추가된 프로 플랜을 제공할 예정이에요.        |
| 어떤 언어를 지원하나요?          | 현재 한국어를 가장 잘 지원해요. 영어도 사용 가능합니다.                                       |
| 녹음은 얼마나 길게 할 수 있나요? | 한 번에 최대 5분까지 녹음할 수 있어요. 3분이면 충분히 정리돼요.                               |
| 내 음성 데이터는 안전한가요?     | 녹음 파일은 안전하게 암호화되어 저장되며, 본인만 접근할 수 있어요.                            |
| 앱을 설치해야 하나요?            | 아니요, 모바일 웹 브라우저에서 바로 사용할 수 있어요. 홈 화면에 추가하면 앱처럼 쓸 수 있어요. |

## 4. 비주얼 디자인 방향

### 컬러 팔레트

| 용도         | 컬러                                        |
| ------------ | ------------------------------------------- |
| Primary      | #6366F1 (Indigo 500) — 신뢰감 + 테크 감성   |
| Primary Dark | #4F46E5 (Indigo 600) — hover/active 상태    |
| Accent       | #F59E0B (Amber 500) — CTA 강조, 포인트 컬러 |
| Background   | #FAFAFA — 밝고 깨끗한 배경                  |
| Surface      | #FFFFFF — 카드/섹션 배경                    |
| Text         | #111827 (Gray 900) — 본문                   |
| Text Muted   | #6B7280 (Gray 500) — 보조 텍스트            |

### 타이포그래피

| 용도        | 폰트                                | 사이즈 (모바일 / 데스크탑) |
| ----------- | ----------------------------------- | -------------------------- |
| 히어로 카피 | Pretendard Bold                     | 28px / 48px                |
| 섹션 제목   | Pretendard SemiBold                 | 22px / 32px                |
| 본문        | Pretendard Regular                  | 16px / 18px                |
| 서비스명    | Pretendard Bold 또는 별도 로고 폰트 | —                          |

### 전체 무드

- **미니멀 & 클린**: 여백 넉넉하게, 요소 간 간격 충분히
- **모바일 퍼스트**: 모바일 사용자가 주 타겟이므로 모바일 레이아웃 우선 설계
- **부드러운 곡선**: border-radius 넉넉하게 (12~16px), 딱딱한 직선보다 부드러운 느낌
- **가벼운 그림자**: 카드/버튼에 subtle shadow로 깊이감 (shadow-sm ~ shadow-md)

## 5. 반응형 브레이크포인트

| 이름    | 범위       | 레이아웃 특성                 |
| ------- | ---------- | ----------------------------- |
| Mobile  | < 768px    | 1컬럼, 세로 스택              |
| Tablet  | 768~1024px | 2컬럼 가능 (기능 카드 등)     |
| Desktop | > 1024px   | 최대 너비 1200px, 가운데 정렬 |

## 6. 얼리액세스 폼 상세

- **수집 항목**: 이메일 (필수)
- **도구**: Tally 임베드 (무료, 깔끔한 UI)
- **제출 후 메시지**: "신청 완료! 출시되면 가장 먼저 알려드릴게요."
- **데이터 활용**: 출시 시 초대 이메일 발송
