## 원라이너 (Tagline)

> **말하면, 정리된다.**

- 영문: Speak. It's organized.
- 서비스의 핵심 가치를 6글자로 압축
- "말하다" → "정리되다"의 인과 관계가 곧 제품의 약속

---

## 메인 카피 (Hero Copy)

> **그냥 말하세요.**
> **semo가 알아서 정리해 드릴게요.**

- 사용자에게 요구하는 건 "말하는 것" 하나뿐이라는 메시지
- 구조화, 요약, 할 일 추출 등 복잡한 과정은 semo가 처리

---

## 서브 카피 (Supporting Copy)

> 녹음 버튼 하나로 시작하세요.
> AI가 요약, 핵심 포인트, 할 일 목록까지 자동으로 만들어 드립니다.

## 가치 제안 (Value Proposition)

| 사용자 문제                                  | semo의 해결                          |
| -------------------------------------------- | ------------------------------------ |
| 좋은 아이디어가 떠올랐는데 타이핑하기 귀찮다 | 말하기만 하면 끝                     |
| 회의 끝나고 정리할 시간이 없다               | AI가 자동으로 요약 + 액션아이템 추출 |
| 메모를 해도 나중에 다시 안 본다              | 구조화된 노트로 한눈에 파악          |
| 음성 메모를 다시 들으려면 전부 재생해야 한다 | 텍스트 전사 + 핵심만 뽑아서 제공     |

## 톤앤매너

### 성격

- **친근하고 캐주얼**: 존댓말 기본, 하지만 딱딱하지 않게. "~해요" 체
- **간결하고 명확**: 불필요한 수식 없이 핵심만
- **자신감 있는 도우미**: "알아서 해드릴게요" — 사용자가 신경 쓸 필요 없다는 확신

### 지양

- 과도한 기술 용어 (AI, LLM, STT 등을 사용자에게 노출하지 않음)
- 감탄사 남발 ("놀라운!", "혁신적인!")
- 부정적 표현 ("~할 필요 없어요" 대신 "~만 하세요"로)

### 예시

```
(O) "말하면, 정리된다."
(O) "semo가 알아서 정리해 드릴게요."
(O) "3분이면 충분해요."
(X) "최첨단 AI가 음성을 분석하여 구조화합니다."
(X) "혁신적인 음성 메모의 새로운 패러다임!"
```
